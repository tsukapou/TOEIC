# 📊 TOEIC PART5 スコア予測機能について

## 🎯 概要

本アプリケーションは、PART5（30問）の解答結果から、**あなたのTOEICスコアを予測**します。

この予測は、TOEIC公式の配点システムと統計データに基づいています。

## 📈 予測される情報

### 1. PART5予測スコア（/200点満点）

PART5の30問に対する推定得点です。

**計算根拠：**
- TOEICリーディングセクションは495点満点
- PART5は全100問中30問（約30%）
- したがってPART5の配点は約150-200点程度

**スコア換算表：**

|--------|----------------|--------|
| 95%以上 | 180点 | ほぼ満点レベル |
| 90-95% | 170点 | 上級レベル |
| 85-90% | 160点 | 中上級レベル |
| 80-85% | 150点 | 中級レベル |
| 75-80% | 140点 | 中級入門レベル |
| 70-75% | 130点 | 初中級レベル |
| 65-70% | 120点 | 初級上レベル |
| 60-65% | 110点 | 初級中レベル |
| 55-60% | 100点 | 初級レベル |
| 50-55% | 90点 | 基礎レベル |
| 40-50% | 75点 | 基礎入門レベル |
| 40%未満 | 変動 | 要基礎固め |

### 2. リーディングセクション予測（/495点満点）

PART5の結果からリーディングセクション全体のスコアを推定します。

**計算方法：**
```
リーディング予測スコア = PART5スコア × 2.75
```

**根拠：**
- PART5で高得点を取れる人は、他のパート（PART6, PART7）でも比較的良いスコアを取る傾向
- 統計的な相関性を考慮した係数

**例：**
- PART5で150点 → リーディング予測：412点程度
- PART5で120点 → リーディング予測：330点程度
- PART5で90点 → リーディング予測：247点程度

### 3. 総合予測スコアレンジ（/990点満点）

リーディングセクションの予測から、TOEIC総合スコアの範囲を推定します。

**計算方法：**
```
最小予測スコア = リーディング予測 × 0.75
最大予測スコア = リーディング予測 × 1.25
```

**注意点：**
- リスニングセクション（495点）の実力は含まれていません
- あくまで参考値として活用してください
- リスニングが得意な方は、実際のスコアはより高くなる可能性があります

## 🔬 予測の精度について

### 信頼性

✅ **高い信頼性の要素：**
- TOEIC公式の配点比率を参考
- PART5とリーディング全体の相関性を考慮
- 統計的なスコア分布データを活用

⚠️ **注意すべき点：**
- PART6, PART7の実力は反映されません
- リスニングセクションの実力は含まれません
- 30問という少ないサンプル数での推定

### 活用方法

#### ✅ 推奨される使い方

1. **学習の進捗確認**
   - 定期的にテストを受けて進捗を確認
   - スコアの推移をチェック

2. **弱点の発見**
   - 問題タイプ別の正答率を確認
   - 苦手分野を重点的に学習

3. **目標設定**
   - 予測スコアを見て現在地を把握
   - 目標スコアまでの距離を確認

4. **モチベーション維持**
   - スコアアップを視覚的に確認
   - 成長を実感

#### ❌ 避けるべき使い方

1. **絶対的な指標として扱う**
   - あくまで参考値です
   - 実際の試験とは異なる場合があります

2. **1回のテスト結果で判断**
   - 複数回受けて平均を見るのがベター
   - ランダム出題なので難易度にバラつきあり

3. **リスニング力の判断**
   - この予測にリスニングは含まれません
   - 総合スコアは参考程度に

## 📚 スコアアップのための学習法

### PART5で高得点を取るために

#### 1. 品詞問題をマスター（40-50%）
- 最頻出パターンを徹底的に
- 動詞・名詞・形容詞・副詞の見分け方
- 文の構造を理解する

#### 2. 動詞問題を攻略（20-25%）
- 時制の判断（過去・現在・未来）
- 完了形の使い分け
- 受動態のパターン
- 助動詞の後ろは原形

#### 3. 前置詞・接続詞を覚える（15-25%）
- 頻出の慣用表現を暗記
- 前置詞の使い分け
- 接続詞の意味の違い

#### 4. スピードアップ
- 1問20-30秒を目標に
- パターン認識を強化
- 迷ったら次へ進む

### スコア別学習アドバイス

#### 予測スコア 400点未満の方
**焦点：基礎固め**
- 品詞の識別を徹底的に
- 基本的な文法ルールの確認
- 頻出単語の暗記
- 本アプリのTest 1-5を繰り返し

#### 予測スコア 400-600点の方
**焦点：応用力強化**
- 複雑な文構造の理解
- 完了形・受動態の習得
- コロケーションの学習
- 間違えた問題の徹底復習

#### 予測スコア 600-800点の方
**焦点：精度向上**
- 微妙な語法の違いを学習
- 難易度の高い問題に挑戦
- スピードと正確性の両立
- PART6, PART7の強化も

#### 予測スコア 800点以上の方
**焦点：満点を目指す**
- 高度な文法知識の習得
- すべてのパターンを完璧に
- ケアレスミスの防止
- 本番形式での練習
