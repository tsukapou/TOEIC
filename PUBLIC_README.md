## 💡 よくある質問（FAQ）

### Q1: 他のデバイスでも使えますか？

**A**: はい、ただしデータは自動同期されません。各デバイスで独立した学習データが保存されます。同期したい場合は、手動でバックアップ→復元が必要です。

### Q2: データが消える心配はありませんか？

**A**: ブラウザのキャッシュをクリアすると消えます。定期的にバックアップを取得してください（admin-menu.html から可能）。

### Q3: 他の人に見られませんか？

**A**: 同じブラウザ・同じPCを使わない限り、他の人が見ることはできません。管理者機能はパスワード保護されています。

### Q4: オフラインでも使えますか？

**A**: 初回ロード後は完全オフラインで動作します。インターネット接続は不要です。

### Q5: スマホでも使えますか？

**A**: はい、レスポンシブデザインでスマホにも対応しています。
