# 🚀 クイックスタート: ツカサさん専用ダッシュボード

**読了時間**: 3分  
**対象**: 初めてダッシュボードを使う方

---

## 📍 どこにあるの？

ホーム画面を開くと、**一番上**に紫色の大きなカードが表示されます。これが「**ツカサさん専用ダッシュボード**」です！

```
┌────────────────────────────────────┐
│  🎯 ツカサさん専用ダッシュボード     │  ← ここです！
│  あなただけの学習サポート           │
└────────────────────────────────────┘
```

---

## 🎯 何ができるの？（3つの機能）

### 1️⃣ 📅 今日の専用メニュー
**あなただけの学習プランを自動生成！**

```
✅ おはようございます、ツカサさん！
   最近の成長が素晴らしいです！12%もUPしています📈

⏰ 推奨学習時間: 14:00-15:00 (約25分)
💡 午後の時間帯で最もパフォーマンスが良いです

📚 今日のおすすめ:
   🔥 動詞の形 10問 - 苦手克服のチャンス！
   ✨ 前置詞 5問 - 得意分野をキープ
   🎲 ランダム 5問 - 全体的な総復習

🎯 今日の目標: あと1回の学習
   ██████░░░░ 1/2回完了

💬 私からのアドバイス:
   「動詞の形」問題が苦手のようですね。
   1日10分でいいので、コツコツ克服していきましょう！
```

### 2️⃣ 📊 ツカサさんの今
**あなたの学習状態をリアルタイム表示！**

```
今日の調子: 🔥 絶好調
連続学習: 7日 🔥
今日の学習: 2回
今日の正答率: 85%

🎯 目標まで:
   現在の推定スコア: 720点
   目標スコア: 800点
   あと: 80点
```

### 3️⃣ 📈 成長サマリー
**あなたの成長を一目で確認！**

```
📈 最近のトレンド: 上昇中 (+12%)
🎯 最近の正答率: 82%
⭐ 得意分野: 前置詞
💪 強化中: 動詞の形

🔮 このペースなら、28日後に目標達成予定！
```

---

## 🎨 色の意味は？

### 優先度カラー:
- 🔥 **赤色** = 今すぐやるべき！（高優先度）
- ✨ **オレンジ** = 早めにやろう（中優先度）
- 🎲 **青色** = 余裕があれば（低優先度）

### 調子カラー:
- 🔥 **濃い赤** = 絶好調！（80%以上）
- 😊 **緑色** = 好調です（70-80%）
- 😊 **グレー** = 通常（60-70%）
- 😅 **黄色** = 疲れてるかも（60%未満）

## 💡 3ステップで始める

### ステップ1: チェック
朝起きたら、ダッシュボードの「**今日の専用メニュー**」をチェック！

### ステップ2: 実行
推奨時間に、推奨問題から学習スタート！🔥

### ステップ3: 確認
学習後、「**リアルタイム状態**」で自分の調子を確認！

## ❓ よくある質問（3つ）

### Q: 「データ不足」って何？
**A**: 初回起動時や学習回数が少ない時に表示されます。
- **解決策**: Test 1から始めて、3-5回学習すればOK！

### Q: 推奨時間が合わない...
**A**: あくまで**提案**なので、自分のスケジュールを優先してください。
- **ポイント**: でも、時間がある時は推奨時間に学習すると効率UP！

### Q: 「要休憩」が続いている
**A**: 疲れているサインです。
- **対策**: 休憩を取る → 得意分野から始める → 学習時間を短くする

## 📊 データはどこから？

- 📊 **正答率**: カテゴリ別の苦手・得意
- 📈 **トレンド**: 最近5回 vs 過去5回
- 🎯 **進捗**: 目標までの距離

すべて**あなたの実際のデータ**から自動算出！
