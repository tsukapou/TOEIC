# 🎯 ツカサさん専用ダッシュボード - ユーザーガイド

## 📖 概要

「**ツカサさん専用ダッシュボード**」は、あなたの学習データを深く分析し、完全にパーソナライズされた学習プランとサポートを提供する機能です。

---

## 🌟 主な機能

### 1. 📅 今日の専用メニュー

#### あなただけの学習プランを毎日自動生成！

**表示内容:**
- ⏰ **推奨学習時間**: あなたが最もパフォーマンスが良い時間帯を自動判定
- 📚 **今日のおすすめ問題**: 苦手克服・得意維持・総復習の3つを優先度別に提案
- 🎯 **今日の目標**: 残り学習回数と目標正答率を視覚的に表示
- 💬 **秘書からのアドバイス**: あなたの状況に合わせた個別メッセージ

**例:**
```
⏰ 推奨学習時間: 14:00-15:00 (約25分)
💡 午後の時間帯で最もパフォーマンスが良いです

📚 今日のおすすめ:
  🔥 優先: 動詞の形 10問 - 苦手克服のチャンス！
  ✨ 維持: 前置詞 5問 - 得意分野をキープ
  🎲 復習: ランダム 5問 - 全体的な総復習

🎯 今日の目標:
  📝 あと1回の学習
  🎯 正答率75%以上を目指そう
  進捗: ██████░░░░ 1/2回完了
```

---

### 2. 📊 ツカサさんの今（リアルタイム状態）

#### あなたの学習状態を常時モニタリング！

**表示項目:**

| 項目 | 説明 | 例 |
|------|------|-----|
| **今日の調子** | 正答率から自動判定 | 🔥 絶好調（80%以上）<br>😊 好調（70%以上）<br>😊 通常（60-70%）<br>😅 要休憩（60%未満） |
| **連続学習** | 学習ストリーク | 7日 🔥 |
| **今日の学習** | 本日の学習回数 | 2回 |
| **今日の正答率** | 本日の平均正答率 | 85% |
| **目標まで** | 現在の推定スコアと目標スコアの差 | 現在: 720点<br>目標: 800点<br>あと: 80点 |

---

### 3. 📈 成長サマリー

#### あなたの成長をグラフ化！

**表示項目:**

| 項目 | 説明 | 例 |
|------|------|-----|
| **最近のトレンド** | 最近5回と過去5回の比較 | 📈 上昇中 (+12%) |
| **最近の正答率** | 直近5回の平均 | 82% |
| **得意分野** | WeaknessAnalysis連携 | ⭐ 前置詞 |
| **強化中** | 苦手カテゴリ | 💪 動詞の形 |
| **予測** | 目標達成予測日 | 🔮 このペースなら、28日後に目標達成予定！ |

## 🎨 ビジュアルガイド

### カラーコード

#### 優先度カラー:
- 🔥 **高優先度（赤）**: 今すぐやるべき問題
- ✨ **中優先度（オレンジ）**: 早めにやるべき問題
- 🎲 **低優先度（青）**: 余裕があればやる問題

#### 調子カラー:
- 🔥 **絶好調（赤）**: 正答率80%以上 - 絶好調です！
- 😊 **好調（緑）**: 正答率70-80% - 良い調子ですね
- 😊 **通常（グレー）**: 正答率60-70% - 普通の状態
- 😅 **要休憩（黄）**: 正答率60%未満 - 疲れているかも

#### トレンドカラー:
- 📈 **上昇（緑）**: 最近の正答率が5%以上UP
- 📊 **安定（グレー）**: 変化が小さい（±5%以内）
- 📉 **調整中（オレンジ）**: 最近の正答率が5%以上DOWN

## 💡 使い方のコツ

### 1. 毎日チェックする
- **朝**: 「今日の専用メニュー」を確認して学習計画を立てる
- **学習後**: 「リアルタイム状態」で自分の調子を確認
- **夜**: 「成長サマリー」で今日の成果を振り返る

### 2. 推奨時間に従う
- ダッシュボードが提案する時間帯は、あなたの過去の学習データから**最もパフォーマンスが良い時間帯**を自動判定しています
- できるだけこの時間に学習すると効率が最大化されます

### 3. 優先度に従う
- 「今日のおすすめ問題」の優先度は、あなたの苦手・得意・復習タイミングを総合的に分析した結果です
- 🔥 高優先度から順にこなすことで、最も効率的に学習できます

### 4. 調子を見極める
- 「今日の調子」が😅 **要休憩**の場合は、無理せず軽めの問題から始めるか、休憩を取りましょう
- 🔥 **絶好調**の時は、難しい問題にチャレンジするチャンスです！

### 5. 目標を意識する
- 「目標まで」セクションで常に目標との距離を確認
- 「このペースなら○日後に目標達成」という予測を励みに継続学習！

## 🔍 よくある質問（FAQ）

### Q1: ダッシュボードが表示されない
**A**: 以下を確認してください：
1. ページを再読み込み（F5 または Ctrl+R）
2. ブラウザのキャッシュをクリア（Ctrl+Shift+R）
3. JavaScriptが有効になっているか確認

### Q2: 「データ不足」と表示される
**A**: 初回起動時や学習データが少ない場合は「データ不足」と表示されます。
- まずは **Test 1** から始めて、データを蓄積しましょう
- 3-5回学習すると、しっかりした分析結果が表示されます

### Q3: 推奨時間が自分のスケジュールと合わない
**A**: ダッシュボードはあくまで**提案**です。
- 自分のスケジュールを優先してOKです
- ただし、時間がある時は推奨時間に学習すると効率が上がります

### Q4: 「要休憩」が続いている
**A**: 以下を試してみてください：
1. **休憩を取る**: 疲れている可能性があります
2. **軽い問題から**: 得意分野から始めて自信を取り戻す
3. **学習時間を短くする**: 一度に30問ではなく、10問×3セットに分ける

### Q5: 目標達成予測日が「999日後」になる
**A**: 以下の場合に発生します：
1. **トレンドが下降**: 最近の正答率が下がっている
2. **データ不足**: まだ十分なデータが蓄積されていない
- 対策: 継続学習でトレンドを上向きに！

## 📊 データの見方

### トレンド判定ロジック
```
最近5回の平均正答率 vs 過去5回の平均正答率

差が +5% 以上  → 📈 上昇中 (+X%)
差が -5% ~ +5% → 📊 安定
差が -5% 以下  → 📉 調整中 (-X%)
```

### スコア予測ロジック
```
最近5回の平均正答率から TOEIC スコアを算出

予測スコア = 400 + (平均正答率 × 400)

例: 平均正答率 80% の場合
  → 400 + (0.8 × 400) = 720点
```

### 目標達成日予測ロジック
```
週2回学習すると仮定

残りポイント = 目標スコア - 現在スコア
週間改善率 = トレンド改善率 × 2
必要週数 = 残りポイント ÷ 週間改善率
目標達成日 = 必要週数 × 7日

例: 目標800点、現在720点、トレンド+12%の場合
  → (800-720) ÷ (12×2) = 3.33週
  → 3.33 × 7 = 24日後
```
