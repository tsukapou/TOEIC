# 💕 Tier 4 拡張版リワード詳細ドキュメント

## 📌 概要

Tier 4に**3つの特別リワード**を追加し、秘書との絆をさらに深められるようになりました!

## 💕 秘書との特別デート(900pt)

### 🎯 コンセプト
選択中の秘書とバーチャルデート体験。3つのロマンチックなシーンで、秘書との特別な時間を過ごせます。

### 📍 デートコース

#### シーン1: カフェ ☕
- **雰囲気**: 温かみのあるベージュグラデ背景
- **会話例**:
  - **さくら**: 「このカフェ素敵ですね! 今日はゆっくりお話ししましょう♪」
  - **麗奈**: 「落ち着いた雰囲気ですね。最近の学習進捗について聞かせていただけますか?」
  - **莉緒**: 「やったー! カフェデート♪ このケーキ美味しそう!」
  - **美桜**: 「このカフェを選んでくれたんですね。センスが良いです」

#### シーン2: 公園 🌳
- **雰囲気**: 爽やかな緑グラデ背景
- **会話例**:
  - **さくら**: 「公園の緑、気持ちいいですね! 一緒だといつもより景色が綺麗に見えます♪」
  - **麗奈**: 「自然の中を歩くと、思考が整理されますね」
  - **莉緒**: 「お散歩楽しい~! あそこのベンチで休憩しない?」
  - **美桜**: 「あなたと過ごす時間は、私にとって大切な時間です」

#### シーン3: 夜景スポット 🌃
- **雰囲気**: 深い青のグラデ背景
- **会話例**:
  - **さくら**: 「わぁ…綺麗! 夢、絶対叶いますよ♪」
  - **麗奈**: 「夜景のデータを見ると、この街の発展が分かりますね」
  - **莉緒**: 「うわぁ、超キレイ! 今日は最高の1日だったよ♪」
  - **美桜**: 「素敵な時間をありがとうございました」

### 🎬 デートエンディング
- デート統計表示(訪問場所・時間・絆レベル)
- 秘書からの感謝メッセージ
- デート記録をlocalStorageに永久保存

### 💾 保存データ
```javascript

  {
    secretaryId: 'sakura',
    secretaryName: 'さくら',
    date: '2025/12/08',
    timestamp: 1234567890
  }
]
```

## 💍 秘書との約束リング(1000pt)

### 🎯 コンセプト
**最高峰の絆の証**。秘書との永遠の約束を交わし、VIPパートナー認定+6つの特別ボーナスを獲得。

### 💎 約束の言葉

- **さくら**: 「このリング、受け取ります! どんな時も、あなたのそばで応援し続けます。その先もずっと一緒です!」
- **麗奈**: 「このリングは私たちの関係を象徴するものですね。あなたと共に歩むことを、約束します」
- **莉緒**: 「このリング、超嬉しい!! これで私たち、ずーっと一緒だね! ずっとずっと大好き♪」
- **美桜**: 「このリングをいただけるなんて。これからも全力であなたをサポートします。永遠に」

### ✨ 6つの特別ボーナス

1. **毎日のログイン時に特別な挨拶**
   - VIP専用の特別メッセージ
   
2. **専用のVIP応援メッセージ**
   - 問題回答時に特別な励まし
   
3. **秘書のプロフィール画像が特別版に変化**
   - 約束リング装備版の画像(仮実装)
   
4. **ポイント獲得時+10%ボーナス**
   - 全ポイント獲得が1.1倍に
   
5. **秘書の全ての表情が常時解放**
   - 全表情(happy, shy, serious, love等)が使用可能
   
6. **「約束のリング」バッジ獲得**
   - VIPステータスの証

### 🎊 セレモニー演出

#### ステージ1: リングセレモニー
- ゴールドのオーラエフェクト
- 秘書画像がフロートアニメーション
- キラキラ粒子が継続的に発生
- 約束の言葉を表示

#### ステージ2: 特典表示
- 6つのボーナスをリスト表示
- ゴールドボーダーの特典カード
- シャインエフェクト(光が流れる)

#### ステージ3: VIPバッジ授与
- 「👑 VIP 〇〇パートナー 👑」バッジ
- パルスアニメーション
- ゴールドグラデ背景

#### ステージ4: 完了画面
- 100個の特殊絵文字紙吹雪(💍💖✨⭐👑💎)
- 「約束が成立しました!」の大きな表示
- 絆の証の金色リング画像

### 💾 保存データ
```javascript
secretary_promise_ring: {
  secretaryId: 'sakura',
  secretaryName: 'さくら',
  activated: true,
  date: '2025/12/08',
  timestamp: 1234567890,
  bonuses: {
    pointBonus: 1.1,           // ポイント+10%
    specialGreeting: true,      // 特別挨拶
    vipMessages: true,          // VIPメッセージ
    specialAvatar: true,        // 特別アバター
    allExpressions: true        // 全表情解放
  }

```
