# 🎀 3人の専属秘書チーム

## 📋 概要

TOEIC PART5 学習アプリに、**3人の個性豊かな専属秘書**が実装されました！✨

あなたの好みや学習スタイルに合わせて、秘書を自由に選択できます。それぞれ異なる性格と指導スタイルで、あなたの学習を全力でサポートします！

## 🔄 秘書の切り替え方法

### 操作手順

1. **👥ボタンをクリック**
   - 画面右下の秘書アバターの上にある「👥」ボタンをクリック

2. **秘書選択パネルが表示**
   - 3人の秘書が表示されます
   - 現在選択中の秘書には「✓」マークが付いています

3. **好きな秘書を選択**
   - 選びたい秘書のカードをクリック
   - 秘書が切り替わり、挨拶メッセージが表示されます

4. **選択が保存される**
   - 選んだ秘書は自動的に保存されます
   - 次回アプリを開いたときも同じ秘書が表示されます

## ⚙️ 技術仕様

### データ保存
- **LocalStorage**を使用
- キー: `toeic_selected_secretary`
- 値: `sakura` / `reina` / `yui`

### 秘書データ構造
```javascript
{
  id: 'sakura',
  name: 'さくら',
  personality: 'gentle',
  description: '優しくバランスの取れたサポート',
  color: '#ec4899',
  avatarUrl: '画像URL'
}
```

### メッセージ生成
- 性格（personality）に応じて異なるメッセージを生成
- `gentle`, `strict`, `cute` の3パターン
- スコアや進捗に応じて動的に変化
